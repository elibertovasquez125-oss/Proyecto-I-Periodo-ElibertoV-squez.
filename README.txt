Requisitos para el uso del sistema:
-------------------------------------------------

Tener instalado:

1. Python 3.12
2. Flask
3. Openpyxl

Instalar las librerías con los siguientes comandos:

pip install flask
pip install openpyxl

si da error también se puede hacer uso de una carpeta
que se puede descargar llamada venv que tiene las
librerias

¿Cómo ejecutar el sistema correctamente?
-------------------------------------------------

1. Abrir una terminal.
2. Ubicarse en la carpeta del proyecto.
3. Ejecutar la aplicación:

python app.py

4. Abrir el navegador.

5. Ingresar a la dirección

6. Iniciar sesión con las los datos correctos.

-------------------------------------------------
Funciones que tiene el sistema internamente:
-------------------------------------------------

- Inicio de sesión.
- Agregar contactos.
- Buscar contactos por nombre.
- Mostrar todos los contactos.
- Ordenar contactos alfabéticamente.
- Editar contactos.
- Eliminar contactos.
- Ver información detallada de contactos.
- Generar reporte de contactos.
- Guarda los datos en Excel por medio de openpyxl.

-------------------------------------------------
Dato extra del sistema para funcionamiento:
-------------------------------------------------

El archivo contactos.xlsx debe estar en la misma
carpeta donde esta app.py para que funcione
correcto funcionamiento del sistema.
